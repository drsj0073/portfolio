About Me

I'm John Dirstein - a front-end web developer from Toronto, Ontario. When I'm not pushing myself in Jiu Jitsu or playing with my cat (Freddie), I spend my time developing professional websites.

I've completed the part time intro and advanced web development at HackerYou, which led me to completing the full time bootcamp shortly after. During this time, I learned to:

  • Build beautiful, responsive websites with SCSS/PostCSS.
  • Utilize JQuery to add to the users experience and create an engaging experience.
  • Develop using ES6 and modular components to maximize organization and readability.
  • Extract data from API's.
  • Create usable applications with React and Firebase which have authorization and a functional database.
